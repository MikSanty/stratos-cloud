# Brand Enterprise Cloud

A modern enterprise-focused marketing website built with Next.js App Router.

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- GSAP animations
- TypeScript

## Project Sections

The homepage is composed of reusable sections:

- Hero
- Features
- Solutions
- Workflow
- Company
- Pricing
- Testimonials
- Contact
- Footer

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` - Start the local dev server
- `npm run build` - Build for production
- `npm run start` - Run the production build
- `npm run lint` - Run ESLint checks

## Deployment

Deploy with any Next.js-compatible platform (Vercel recommended).
